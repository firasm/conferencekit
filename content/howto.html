Adding images¬∂

  1. Upload the image to the GitHub repository. You can do this by dragging and dropping the image to the content/images folder on GitHub.

  2. Note the filename of the image, mine is called ‚ÄúdemoImage.jpg‚Äù.

  3. Add the image to the Website like this:

![](images/demoImage.jpg)

  1. Remember to always include alt-text with images, for the benefit of those who cannot see the images (or anyone working with a browser that isn‚Äôt loading images).

  2. (Optional) By default, the image will be the same size you upload it. To change the width or the height, instead of using ![](content/images/demoImage.jpg) you will need to use this syntax:

<img src="images/demoImage.jpg" width=50>

Warning

When adding images to .md files inside the ‚Äúcontent‚Äù folder on GitHub, your path (i.e. images/demoImage.jpg) will need to have an additional ../ so the full code would look like ![](../content/images/demoImage.jpg)).
